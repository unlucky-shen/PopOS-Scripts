# PopOS-Scripts

Quick PopOS setup scripts incase i have to use it,

Make the scripts executable first,
```bash
chmod +x *.sh
```

execute scripts using Bash,
```bash
bash <script_name>.sh
```

Scripts execution order for smooth execution,
```bash
devtools.sh
kittysetup.sh
addrepos.sh
flatpak.sh
debloat-and-install.sh
ssh-git.sh
gitclone.sh
```
